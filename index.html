核心功能与技术优势

极致速度连点

ClickerFree 采用低级输入模拟库 **pynput**，并结合 **PyQt5 线程** 独立运行核心任务。这保证了：

  • **毫秒级响应：** 最低支持 0.001s (1 毫秒) 的执行间隔，逼近系统限制，适用于对速度要求极高的场景。
  • **UI 不卡顿：** 任务在后台线程高效运行，主界面和日志输出保持流畅响应。

智能宏脚本录制

宏录制功能不仅限于简单的点击，而是**完整捕捉**您的操作路径：

  • **全方位记录：** 记录鼠标移动轨迹、左右键点击、拖拽操作，以及键盘上的所有输入和按键。
  • **时间精准同步：** 精确记录您每次操作之间的真实延迟，实现高保真回放。
  • **可编辑性：** 录制的宏可导出为 **JSON 文件**，方便您手动修改时间间隔、加速回放，或调整特定操作坐标。

强大的全局热键控制

通过全局监听器，您可以在任何应用程序或游戏窗口中，快速、无干扰地控制 ClickerFree 的状态：

  • **运行/停止:** 随时按下 F8 键，立即启动或暂停当前设置的任务（连点、输入或宏回放）。
  • **录制/停止:** 按下 F10 启动宏录制，再次按下 F10 即可停止并捕获脚本。

键盘输入与循环

专为需要重复输入文本的场景设计，提供灵活的输入循环控制：

  • **自定义文本：** 循环输入您设定的任何文字或字符串。
  • **高级选项：** 可选择在每次输入结束后自动添加**空格**或**回车（Enter）**键，简化表单填写或聊天操作。

源码运行与环境配置

作为开源项目，欢迎开发者从源码运行或贡献代码。您需要安装以下依赖：

安装必需的 Python 依赖

pip install PyQt5 pynput mouse keyboard

运行程序

python clickerfree.py
